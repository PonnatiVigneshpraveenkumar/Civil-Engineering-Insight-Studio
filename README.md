# 🏗️ Civil Engineering Insight Studio

AI-powered structural analysis and engineering insights from images using Google Gemini AI and Streamlit.

---

## 🚀 Overview

Civil Engineering Insight Studio is a web application that allows users to:

- Upload images of civil structures
- Analyze structural elements
- Identify materials used
- Get safety and maintenance insights
- Receive AI-powered engineering observations

Built with:
- Streamlit
- Google Gemini API
- Python
- PIL (Image Processing)

---

## 🖼️ Features

- 📷 Image upload and preview
- 🧠 AI-based structural analysis
- 📊 Engineering insights output
- 🖥️ Clean dashboard layout with right-side control panel
- ☁️ Deployable on Vercel / Streamlit Cloud

---

## 🛠️ Tech Stack

- Python 3.9+
- Streamlit
- Google Generative AI (Gemini)
- python-dotenv
- Pillow
